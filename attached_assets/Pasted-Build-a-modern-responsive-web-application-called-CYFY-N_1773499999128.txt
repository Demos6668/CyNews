Build a modern, responsive web application called "CYFY News Board" - an internal cybersecurity threat intelligence, advisories, and news aggregation platform. The app should have two main segments: Local Threats/News and Global Threats/News.

=== TECH STACK ===
- Frontend: React.js with Vite
- Styling: Tailwind CSS (custom theme)
- Backend: Node.js with Express
- Database: SQLite
- State Management: Zustand
- Icons: Lucide React
- Date Handling: date-fns

=== CORE FEATURES ===

1. DASHBOARD (HOME)
- Threat Level Indicator: Visual gauge showing current threat level (Low/Medium/High/Critical) with animated gradient using teal-to-amber-to-red
- Stats Cards: Total threats today (local/global), Active advisories count, Critical alerts needing attention, Resolved incidents
- Recent Activity Feed: Live-updating timeline of latest entries
- Quick Filters: Toggle between Local/Global view

2. LOCAL NEWS & THREATS SECTION
- Filter by: Region, Industry Sector, Threat Type, Severity, Date Range
- Categories: Regional cyber attacks, Local infrastructure threats, Country-specific malware campaigns, National CERT advisories, Local compliance updates
- Card display with: Title, Severity Badge, Source, Timestamp, Brief Summary
- Click to expand full details in modal/slide-over

3. GLOBAL NEWS & THREATS SECTION
- Filter by: Region, Attack Vector, Threat Actor, Severity, Date Range
- Categories: Global ransomware campaigns, APT group activities, Zero-day vulnerabilities, International CERT bulletins, Major breach disclosures
- Same card structure as local but with geographic tags

4. ADVISORIES SECTION
- Security advisories from: CISA, NIST, CVE Database, Vendor bulletins
- CVSS Score display with color coding
- Affected products/versions listing
- Mitigation steps accordion
- Status: New / Under Review / Patched / Dismissed

5. THREAT INTELLIGENCE FEED
- IOC (Indicators of Compromise) listings
- TTPs (Tactics, Techniques, Procedures)
- Threat actor profiles
- Campaign tracking
- Export functionality (CSV/JSON)

=== DATA MODELS ===

Threat/News Item:
{
  id: string,
  title: string,
  summary: string,
  content: string,
  type: 'threat' | 'news' | 'advisory',
  scope: 'local' | 'global',
  severity: 'critical' | 'high' | 'medium' | 'low' | 'info',
  category: string,
  source: string,
  sourceUrl: string,
  region: string[],
  tags: string[],
  iocs: string[],
  affectedSystems: string[],
  mitigations: string[],
  publishedAt: timestamp,
  updatedAt: timestamp,
  status: 'active' | 'resolved' | 'monitoring'
}

Advisory:
{
  id: string,
  cveId: string,
  title: string,
  description: string,
  cvssScore: number,
  severity: string,
  affectedProducts: string[],
  vendor: string,
  patchAvailable: boolean,
  patchUrl: string,
  workarounds: string[],
  references: string[],
  publishedAt: timestamp
}
